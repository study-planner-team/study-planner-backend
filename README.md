# Study Planner Backend

The Study Planner Backend is a RESTful API service designed to manage study plans, topics, materials, sessions, quizzes, and user statistics. It serves as the core of the Study Planner application, providing essential backend functionality and data management.
## Features

- **User Management:** Registration, authentication, and profile management.
- **Study Plans:** Create and manage personalized study plans.
- **Topics & Materials:** Organize study topics and associated materials.
- **Study Sessions:** Track and log study sessions.
- **Quizzes:** Create and manage quizzes for self-assessment.
- **Statistics:** Monitor study progress and performance metrics.

## Technologies Used

- **Backend Framework:** .NET Core Web API
- **Database:** Azure SQL Database
- **Authentication:** JWT-based authentication
- **CI/CD:** GitHub Actions
- **Deployment:** Azure Web App
